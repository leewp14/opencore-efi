This option runs four commands.

- First, /usr/bin/uname is run to dump the kernel version and architecture.
- Second, /usr/sbin/sysctl is run to dump the cpu information.
- Third, /usr/sbin/sysctl is run to dump the hardware information.
- Forth, /usr/sbin/sysctl is run to dump the xcpm information.

Each dump is written to it's own file and saved in the latest dump/Kernel folder.
