This option performs two steps.

First, for a complete full IORegistry dump, the ioregwv tool is run five times, once for each IORegistry plane. A shell script is then run to process the output and build an IORegViewer folder in the latest dump/IORegistry folder.

Source ioregwv from blackosx repository

Second, the /usr/sbin/ioreg tool is run to dump the IOService plane to a file named IOReg.txt and the IODeviceTree plane to a file named IORegDT.txt. Each file is then saved in the latest dump/IORegistry folder.
