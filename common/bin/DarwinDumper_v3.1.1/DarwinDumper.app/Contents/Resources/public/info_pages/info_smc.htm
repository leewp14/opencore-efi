The System Management Controller (AppleSMC) chip uses 4-character keys to represent various data, such as fan speeds and temperatures.

This option runs the smcutil tool to dump all keys and their current values reported by the current system, and then the SMC_util_FansOnly tool to dump all fans.

Source smcutil (Dropbox Link)
Source SMC_util_FansOnly (Dropbox Link)

Two files are produced, SMCkeys.txt and SMC-fans.txt which are saved in the latest dump/SMC folder.
